This is my Pathfinder project, I created this as it looked cool to make, I followed a tutorial from the stasis started projects.
My project is a pathfinder that has three buttons a XIAO microcontroller and three LEDS, the purpose of this was to have fun and having a good experience. One of my functions was that if you click the buttons in the right order one of the lights turns on and for three times
